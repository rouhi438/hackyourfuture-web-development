# 🧠 Memory Card Game

A browser-based **Memory Matching Game** built using **Vanilla JavaScript, HTML, and CSS**.

---

## 🌐 Live Demo

👉 Play The Game [Here](https://rouhi438.github.io/hackyourfuture-web-development/foundation-project/)

---

## 🎮 Game Description

This Memory Game is designed to test player memory and concentration.

Players flip two cards at a time and try to match pairs until the board is completed.

---

## ✨ Features

### 🎯 Core Gameplay

- Card flip animation
- Match detection system
- Board lock while checking cards
- Auto reset after wrong match
- Win detection

---

### ⏱ Timer System

- Normal Mode → Timer counts up
- Hard Mode → Countdown timer (40 seconds)
- Low time warning color

---

### 🎮 Game Modes

#### Normal Mode

- No time limit
- Relaxed gameplay

#### Hard Mode

- 40 seconds time limit
- Game ends if time reaches zero

---

### 📊 Game Stats

- Move Counter
- Matched Pairs Counter
- Timer Display

---

### 🔊 Sound Effects

- Flip sound
- Match success sound
- Wrong match sound
- Shuffle sound
- Winner sound
- Game over sound
